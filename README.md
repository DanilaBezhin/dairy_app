# 📔 Личный Дневник

Минималистичный электронный дневник с premium dark-дизайном.
Стек: **Flask + SQLite3** (без ORM, чистый Python).

## 🚀 Запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить
python app.py
```

Открыть в браузере: **http://localhost:5000**

## ✨ Функционал

- ✅ Создание / редактирование / удаление записей
- ✅ Поиск по заголовку, тексту и тегам (с подсветкой)
- ✅ Сортировка по дате (новые / старые)
- ✅ Настроение к каждой записи (5 вариантов)
- ✅ Теги для категоризации
- ✅ Статистика (всего / за месяц)
- ✅ Горячие клавиши: Ctrl+S, Ctrl+N, Esc
- ✅ Счётчик символов в редакторе

## 📁 Структура

```
diary/
├── app.py              # Flask приложение + API
├── diary.db            # SQLite база (создаётся автоматически)
├── requirements.txt
├── templates/
│   └── index.html      # SPA шаблон
└── static/
    ├── css/style.css   # Premium dark дизайн
    └── js/app.js       # Frontend логика
```

## 🔌 API

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/notes` | Список записей (params: search, sort) |
| POST | `/api/notes` | Создать запись |
| GET | `/api/notes/:id` | Одна запись |
| PUT | `/api/notes/:id` | Обновить запись |
| DELETE | `/api/notes/:id` | Удалить запись |
| GET | `/api/stats` | Статистика |
